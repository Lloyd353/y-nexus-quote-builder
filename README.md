# y-nexus-quote-builder
C' est un outil d'ingénierie financière conçu pour structurer les propositions commerciales dans les secteurs de l'énergie, des télécoms et des automatismes. Il permet aux ingénieurs et chargés d'affaires de générer des devis techniques et des notes d'estimation budgétaire standardisés, avec export PDF professionnel, en moins de deux minutes.
